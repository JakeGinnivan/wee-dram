# Boilerplate: Knex.js

Phase 1 boilerplate project with:

 - Express
 - Knex
 - SQLite3
 - Express Handlebars
 - body-parser
 - AVA
 - Node Inspector
 - nodemon


## Install

```
npm install
npm run knex migrate:latest
npm run knex seed:run
npm run dev
```

Be sure to check out the other npm scripts too.
